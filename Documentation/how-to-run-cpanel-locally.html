How to run cPanel project locally?

If you don't want to run the cPanel project locally, you can skip this section.

cPanel-project

These things are required, to run the project.

  • XAMP
  • PHP 7.2+
  • Node js 12+

Installing PHP in windows

  1. Download the latest version of PHP from the official website ( https://windows.php.net/download/ ). Make sure to select the VC15 x64 Non Thread Safe version if you're using a 64-bit version of Windows.
  2. Extract the contents of the ZIP file to a directory on your system, such as "C:\php".
  3. Open the "Environment Variables" dialog in Windows by opening the Start menu, typing "Environment Variables" and clicking on the "Edit the system environment variables" option.
  4. In the System Properties window, click on the "Environment Variables" button.
  5. Scroll down to the "System Variables" section and find the "Path" variable. Click on it and then click on the "Edit" button.
  6. Add the path to your PHP installation to the "Path" variable, such as "C:\php".
  7. Open the command prompt and type "php -v" to verify that PHP is installed and working correctly.

Installing PHP in Mac

  1. Open Terminal and type the following command to check if PHP is already installed on your Mac:
    2. php -v
  2. If PHP is already installed, you will see its version number. If it is not installed, you can proceed with the following steps.
  3. Use Homebrew to install PHP:
    4. brew install php
  4. Verify that PHP is installed by running the following command:
    5. php -v

Installing PHP in linux

  1. Open a terminal and use the package manager for your Linux distribution to install PHP. The exact command will depend on the distribution you are using.
  2. Here are a few examples:

    For Ubuntu and Debian-based systems:

    sudo apt-get install php

    For CentOS and Fedora-based systems:

    sudo yum install php

  3. Verify that PHP is installed by running the following command:

    php -v

Database

  1. Start the XAMPP control panel and make sure that Apache and MySQL are running.
  2. Open a web browser and navigate to http://localhost/phpmyadmin/. This will take you to the PHPMyAdmin interface.
  3. In the PHPMyAdmin interface, click on the "Databases" tab.
  4. In the "Create database" section, enter a name for your database and select the appropriate character set and collation. Then, click on the "Create" button.
  5. Your new database should now appear in the left-side column of the PHPMyAdmin interface. Click on the name of the database to access its structure.
  6. To import a database, click on the "Import" tab in the top navigation menu.
  7. In the "File to import" section, click on the "Choose File" button and select the SQL file you want to import.
  8. If the file is compressed, select the appropriate compression format from the "Format" dropdown menu.
  9. In the "Character set of the file" section, select the appropriate character set.
  10. Finally, click on the "Go" button to start the import process.
  11. Once the import process is complete, you should see a message indicating that the import was successful. You can now access and work with your imported database in PHPMyAdmin.

Note: These steps are intended as a general guide and may vary slightly depending on the version of XAMPP that you are using. If you run into any difficulties, you may want to consult the XAMPP or PHPMyAdmin documentation

Configure Backend/Admin panel

  1. Navigate to the root directory of cPanel-project folder.
  2. Go to backend folder
  3. Open .env in a text editor
  4. Change the value of APP_URL to APP_URL=http://127.0.0.1:8000
  5. Change the value of CLIENT_BASE_URL to CLIENT_BASE_URL=http://127.0.0.1:3000
  6. Change the value of DB_DATABASE. Put here the name the database you created from the last step.

Run Backend/Admin panel

  1. Navigate to the root directory of cPanel-project > backend folder using the terminal.
  2. Run the following command to start the built-in PHP web server:
    3. php artisan serve
  3. This will start the web server and make your application accessible at http://localhost:8000. You should see a message in the terminal indicating that the server is running, and you can access your application by visiting http://localhost:8000 in a web browser.
  4. If you want to run the server on a different port, you can specify the port number after the serve command:
    5. php artisan serve --port=8080

    This will start the web server and make your application accessible at http://localhost:8080.

Clearing cache

Now it's time to clear the cache of the backend.

  1. Open http://127.0.0.1:8000/api/clear-cache in the browser. Cache has been cleared.
  2. You can browse the admin panel now from http://127.0.0.1:8000.
  3. Use these credentials below to login to the admin panel
    • Admin email: admin@mail.com

Configure Frontend/Website

  1. Navigate to the root directory of cPanel-project folder.
  2. Go to frontend folder
  3. Open config.json file in a text editor.
  4. Change the value of apiBase. Put here the url of the backend from last step. eg. http://127.0.0.1:8000/

Run Frontend/Website

  1. Navigate to the root directory of cPanel-project > frontend folder using the terminal.
  2. Run the following command to install all the packages:
    3. npm i
  3. Run the following command to start the web server in development mode:
    4. npm run dev
  4. You can browser the website now. Website will be run in 3001 port. Browse the website in http://localhost:3001/.
